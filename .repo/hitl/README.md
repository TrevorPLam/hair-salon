# HITL Items Directory

**File**: `.repo/hitl/README.md`

This directory contains individual HITL (Human-In-The-Loop) item files.

## Format
Each HITL item is stored as `HITL-XXXX.md` where XXXX is a sequential number.

## Index
See `.repo/policy/HITL.md` for the index of active and archived HITL items.

## File Naming
- Format: `HITL-0001.md`, `HITL-0002.md`, etc.
- Sequential numbering ensures uniqueness
- Leading zeros for sorting

## Status
- Active items are tracked in `.repo/policy/HITL.md` Active table
- Completed/Superseded items are moved to Archived table
- Item files remain in this directory for historical reference
